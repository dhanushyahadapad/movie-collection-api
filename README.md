Movie Collection Display API
About the Collection

This project includes a small set of classic and modern movies across multiple genres. The collection is designed to be simple, fun, and easy to browse through both API endpoints and a basic frontend page.

Project Description

A lightweight Node.js + Express API that serves movie data from an in-memory array.
A frontend HTML page is included to view all movies, classic movies, and available genres using simple fetch requests.
